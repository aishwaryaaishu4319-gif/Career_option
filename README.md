# Career Compass (Frontend)

Career Compass is a frontend-only career guidance platform that helps students explore careers, roadmaps, assessments, colleges, skill learning, and job market insights.

## Features
- Landing page with search, categories, trending careers, and success stories
- Category pages with related careers
- Detailed career pages with skills, education paths, salaries, and roadmaps
- Career assessment test UI
- College and course finder with filters
- Skill learning module with platforms and courses
- Job market insights with demand and salary trends
- Student dashboard for saved careers and progress

## Getting Started

Install dependencies:
```
npm install
```

Run the development server:
```
npm run dev
```

Build for production:
```
npm run build
```

Lint the project:
```
npm run lint
```
